Run the create_rdoc.bat in the product folder to product the RDocs.
  i.e. For GameStop finders, run the following: 
    C:\dev\AutomationFinders\GameStop\src\create_rdoc.bat

Run rake at the AutomationFinders root to execute unit tests.
  In cmd or powershell enter this command:
    cd C:\dev\AutomationFinders
    rake

  You'll see all the unit tests being executed with results.
